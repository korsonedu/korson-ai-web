# 知行智慧学习系统 (UniMind AI OS) - 核心亮点

> **定义下一代数字化教学基础设施：让 AI 真正懂学生，让教研真正自动化。**

知行智慧学习系统不仅是一个课程平台，更是一个基于数据驱动、AI 增强的深度学习闭环生态。

---

## 🚀 核心支柱 (Core Pillars)

### 1. 具备“上帝视角”的专属 AI 导师
不同于通用的聊天机器人，我们的专属导师深度钩稽了后端学术数据库：
*   **掌握度感知**：AI 潜意识里清楚学生当前的 ELO 积分和全站排名。
*   **错题透视**：当学生说“检查错题”时，AI 能直接报出其最近做错的 5 道原题，并针对性地分析其思维误区。
*   **双轨人格**：支持通用学术助教与专属私人导师自由切换，满足从公共答疑到私教辅导的全场景需求。

### 2. 艾宾浩斯智能天梯 (Adaptive Learning)
*   **动态抽题算法**：基于艾宾浩斯遗忘曲线，精准捕捉学生的“遗忘临界点”，实现查漏补缺。
*   **ELO 学术段位**：引入竞技级 ELO 积分系统，量化每一次进步，建立全站实时学术排名，激发学习动力。

### 3. AI 工业级教研作业中心 (AI Content Studio)
*   **全类型识别**：支持从 Word/文本中一键提取**客观题、计算题（含公式）、简答题及深度论述题**。
*   **分片解析引擎**：支持 5-10 万字超长语料的并行处理，教研效率提升 100 倍。
*   **物理同步机制**：支持提示词模板（Prompt Template）物理备份与双向同步，教研员可零代码调优 AI 逻辑。

### 4. 学术知识拓扑地图 (Knowledge Topology)
*   **多维资源挂载**：视频、文章、题库全量挂载至树状知识节点。
*   **可视化溯源**：点击地图节点，实时呈现关联的所有教学资源，构建网状知识体系而非孤岛。

### 5. 极简工业美学 (Apple-Style UI)
*   **精致布局**：适配专业教研场景的紧凑型管理界面，高屏效展现海量数据。
*   **全沉浸深色模式**：针对深夜学习场景优化的视觉主题，保护视力的同时提升质感。
