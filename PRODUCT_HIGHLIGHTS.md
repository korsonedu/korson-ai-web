# Korson AI 智能学习平台 - 产品功能亮点说明
> *本示例文档旨在辅助营销推广与广告文案撰写，核心突出了“数据驱动”、“高效复习”与“极致体验”。*

## 🚀 核心价值主张
**“拒绝盲目刷题，让每一次练习都精准有效。”**
Korson AI 结合**艾宾浩斯遗忘曲线算法**与**可视化知识图谱**，为用户提供一套科学、高效、沉浸式的学习解决方案。

---

## 🌟 三大核心功能模块

### 1. 全景知识地图 (Smart Knowledge Graph)
**—— 看得见的思维脉络，摸得着的知识体系**
*   **可视化导航**：采用动态力导向图（Force-Directed Graph）技术，将枯燥的知识点转化为生动的交互式图谱，知识关联一目了然。
*   **优化交互体验**：
    *   **平滑缩放**：精心调教的缩放速率，拒绝忽大忽小的突兀感，浏览更丝滑。
    *   **精准点选**：大幅优化元素点击判定区域，操作更精准，所点即所得。
    *   **全局掌控**：支持拖拽与层级查看，帮助用户快速构建宏观知识框架。

### 2. 智能测评系统 (Adaptive Assessment Engine)
**—— 每一道题，都是为你量身定制**
*   **算法驱动复习**：内置**艾宾浩斯记忆算法**，根据用户做题历史智能计算遗忘点，精准推送需要复习的题目，让记忆留存率提升 300%。
*   **极简高效UI**：
    *   **高密度展示**：重新设计的题目展示布局，大幅减少页面无效空白，提升阅读效率。
    *   **专注模式**：每次评估仅展示用户设定的抽题数量（如 5/10/20 题），避免“题海”带来的心理压力。
*   **高质量内容库**：
    *   **严选题库**：内置 **244** 道高质量精选题目，经过严格去重与纠错（已移除历史冗余数据），确保练习质量。
    *   **公式完美渲染**：原生支持 LaTeX 数学公式显示，无论是复杂的微积分还是物理公式，都能清晰呈现。

### 3. 专家级答疑空间 (Expert QA System)
**—— 每一个疑问，都值得被认真对待**
*   **全能提问**：全新重构的答疑系统，支持文本、图片及附件上传，还原真实线下辅导场景。
*   **知识沉淀**：
    *   **星标收藏**：一键收藏经典难题或易错题。
    *   **状态追踪**：清晰标记问题解决状态，打造个人专属的错题/难题库。
*   **稳健架构**：后端数据库经深度优化（FAQ System Refactor），数据存取更安全，响应速度毫秒级。

## 🛠 辅助功能与技术优势
*   **智能导入工具**：提供便捷的题库导入脚本，支持从 JSON 格式一键批量导入试题，轻松扩展教学资源池。
*   **现代化技术栈**：基于 React + Django 构建，兼顾前端的极致流畅与后端的数据安全。
